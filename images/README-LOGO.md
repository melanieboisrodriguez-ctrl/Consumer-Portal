# Logo Brown Shoes

## Emplacement du logo

Le logo Brown Shoes doit être placé dans ce dossier : `images/brown-shoes-logo.png`

## Format recommandé

- **Format** : PNG (avec transparence de préférence)
- **Taille** : Hauteur maximale de 40px (largeur automatique)
- **Nom du fichier** : `brown-shoes-logo.png`

## Comment ajouter le logo

1. Placez votre fichier logo dans le dossier `images/`
2. Nommez-le exactement : `brown-shoes-logo.png`
3. Le logo apparaîtra automatiquement dans tous les mockups sous le numéro de suivi

## Fallback

Si le logo n'est pas trouvé, le texte "Brown Shoes" s'affichera à la place.

## Emplacement dans les mockups

Le logo apparaît juste après le numéro de suivi, avant la boîte d'animation de statut.
